How to Recover Deleted Files!

Deleted your important data by mistake or because of some technical issues & want to recover. Its possible if you have that device & its working!

Let's do it...

1. First Download " Recuva " software by clicking the button down below.

Download!

2. Now Install " Recuva " - If you don't know how to install it steps are given below ->

 

1.  Open the downloaded setup.

2. A dialog box will appear click " yes " in it.

3. Click " Next ".

4. Click " Install ".

5. Wait till it installs.

6. Now uncheck " Release Notes " (if you want read release notes skip this step).

7. Now Click " Finish ".

 

3. Now Recuva icon will appear on your Desktop.

 

4. Now open it.

5. Click " Next ".

6. Select the type of files you want to recover like - photos or video or documents & if you want to recover all types of files select " All Files ".

7.Now choose the place from where you want to recover files like - from "My Documents" or " Recycle Bin " & if you want to recover files from a special folder follow next step -

-> " In a specific location " & follow these steps -

 

- click on " In a specific location ".

- now click on " Browse ".

- a dialog box will appear.

- go to the folder from which you want to recover files (you can open any folder by double clicking).

- now select that folder by a single click & click " OK " .

 

8. Now click " Next ".

9. Now check the " Enable Deep Scan " to search files more deeply.

 

10. Now click start & it will start finding deleted files!

11. Now wait till the scan ends.

12. Now check the files you want to recover

 

Use of colorful Dots -

 

Green - File will recovered without any problem.

 

Yellow - Mostly chances that file will be recovered.

 

Red - Less chances of recovery

13. Now after selecting the files click " Recover " button on the right down corner.

14. Now select the folder in which you want to save the Recovered Files & click " OK ".

 

Note - you can open any folder by double clicking.

 

15. Now it will start recovering wait till the loading ends.

16. now a dialog box will appear with information of files recovered.

 

note - Partly recovered files are not completely recovered files.

 

17. Now check the folder in which you saved the files and your files will be there!

 

18. Its all DONE!
